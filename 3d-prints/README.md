# 3D-printed parts

Both parts in this build come from existing community models — no STLs are re-hosted here. Credit and download links below.

## ESP32-CAM case + mount

**Modular ESP32 CAM Case + Mount** — MakerWorld
<https://makerworld.com/en/models/588781>

Holds the AI-Thinker ESP32-CAM with a modular mount that can be swapped for different bases.

## Wemos 18650 shield case

**Wemos 18650 charge circuit shield case (updated)** — Thingiverse
<https://www.thingiverse.com/thing:5694322>

Snap-fit case for the Wemos 18650 charge-circuit shield + cell.
